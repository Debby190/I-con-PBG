# i-CON PBG - Monitoring Keterlambatan Berkas PBG

Aplikasi Streamlit untuk:
- Pencarian status permohonan
- Monitoring tahapan proses (visualisasi tren bulanan)
- Laporan & ekspor data CSV
- Highlight otomatis keterlambatan berdasarkan SOP

##  Deploy via Streamlit Cloud

1. Fork atau clone repository ini
2. Buka https://share.streamlit.io (Streamlit Cloud)
3. Login dengan GitHub
4. Klik **New App**
5. Pilih repository ini
6. Pilih branch utama (main)
7. Pilih file utama â†’ `app.py`

Aplikasi akan otomatis live dalam beberapa menit.
